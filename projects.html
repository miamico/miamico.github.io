Krylov Quantum Diagonalization

Implements a Krylov-subspace algorithm to extract low-energy spectra of many-body Hamiltonians on IBM’s 127-qubit heavy-hex processors. A shallow Trotter circuit prepares a sample sub-space; classical post-processing then yields eigenvalues that rival exact diagonalisation for 56-site lattices. The technique bridges the gap between variational methods and fault-tolerant phase-estimation.

Paper Code / Tutorial

Tensor-Network-Enhanced Dynamic Multiproduct Formulas

Combines tensor-network compression with high-order multiproduct formulas (MPF) to simulate quantum dynamics more accurately than either approach alone. Expectation values are sampled on hardware; tensor networks compute optimal MPF coefficients offline, reducing algorithmic error an order of magnitude.

Paper Code ( qiskit-addon-mpf )

Probabilistic Error Cancellation & Amplification

Qiskit’s PEC/PEA toolkit models device noise via local process tomography and reconstructs an unbiased estimator of the ideal circuit. The package streamlines calibration, quasi-probability computation, and shot-frugal resampling, allowing users to swap between zero-noise extrapolation and PEC without rewriting workloads.

Paper Code / Docs

Noise-Aware Qubit Selection

A runtime-adaptive heuristic that chooses qubits with the lowest aggregate error rates (T1, T2, readout, crosstalk) just before job submission. Integrated into the IBM Quantum Learning path, it demonstrates up to 2× fidelity gain on depth-40 random circuits without extra calibration overhead.

Paper Tutorial

Context-Aware Compiling

A transpiler pass that analyses the gate-by-gate “context” of each qubit layer and inserts targeted dynamical-decoupling pulses only where they mitigate correlated noise. Demonstrated an 18 % layer-fidelity boost on 10-qubit circuits and released as ContextAwareDynamicalDecoupling in Qiskit Terra 0.46.

Paper Code / API

Error-Robust Gate-Set Design (Deep RL)

Uses deep reinforcement learning to discover microwave pulse shapes that halve two-qubit error rates while shortening gate time. The method autonomously adapts to slow drift and requires no analytic device model, paving the way for AI-driven calibration of large qubit registers.
